## Project Overview
Luminous Tracer is a visual code execution tracer SPA. Users paste or select code snippets in JS/Python/Java/C/C++/Pseudocode, hit Run, and step through animated execution frames — seeing variables, call stacks, execution trees, and flow timelines update in real time.

## Tech Stack
- React 18 + TypeScript (Vite)
- Zustand (state: frames, currentStep, language, status, breakpoints)
- CodeMirror 6 (editor)
- Framer Motion (panel animations)
- Custom JS interpreter (acorn AST) + Pyodide worker for Python
- Google Fonts: Space Grotesk (headlines), Inter (UI), JetBrains Mono (code), Material Symbols Outlined (icons)
- CSS custom properties (no Tailwind — all inline styles + CSS vars in `index.css`)

## Architecture
`App.tsx` is the single layout shell: fixed header (TopAppBar), collapsible left sidebar (SideNavBar with 4 nav items + language selector), main canvas (header metrics row + CodeEditor + floating glass inspector panel), and a bottom footer (FlowTracePanel + ExecutionTimeline + scrub bar).

State lives in `src/store/executionStore.ts` via Zustand. The engine (`src/engine/`) runs code and returns `Frame[]` objects. Each frame has `{type, line, variables, callStack, error, isBreakpoint}`.
